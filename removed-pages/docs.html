Quick Start Guide

Get Started in 5 Minutes

The Blaze Intelligence API provides access to real-time sports data, advanced analytics, and machine learning insights. Built on ESPN endpoints with our proprietary enhancements.

  1. Sign up for a free API key at developer portal
  2. Install the SDK for your preferred language
  3. Authenticate your first request
  4. Fetch live game data or historical stats
  5. Build your championship application
JavaScript
// Initialize the Blaze Intelligence SDK
const blaze = new BlazeAPI({
  apiKey: 'your-api-key',
  environment: 'production'
});

// Get live game data
const liveGames = await blaze.games.getLive({
  league: 'NFL',
  include: ['score', 'analytics', 'predictions']
});

// Access Blaze-specific metrics
const qbrx = await blaze.players.getMetric({
  playerId: 'mahomes-patrick',
  metric: 'QBR-X',
  season: '2024'
});
