Core Endpoints

GET /api/v2/teams/{teamId}/analytics

Retrieve comprehensive analytics for a specific team

# Example Request

GET https://api.blaze-intelligence.pages.dev/v2/teams/cardinals/analytics

# Example Response

{
  "teamId": "cardinals",
  "readinessIndex": 95.2,
  "performanceMetrics": {
    "offense": 94.8,
    "defense": 88.3,
    "specialTeams": 92.1
  },
  "timestamp": "2025-08-31T15:30:00Z"
}
