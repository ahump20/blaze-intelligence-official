EWMA Projection Algorithm

Our Exponentially Weighted Moving Average (EWMA) model for game score projections uses a transparent, reproducible algorithm:

function ewma(values, alpha = 0.35) { if (!values || values.length === 0) return null; let result = null; for (const value of values) { if (value === null || value === undefined) continue; result = result === null ? value : alpha * value + (1 - alpha) * result; } return result; }

Parameters:
• Alpha (α) = 0.35 (weights recent games more heavily)
• Time window = Last 10 completed games
• Data source = MLB StatsAPI (public data)
• Update frequency = Real-time with game completion
